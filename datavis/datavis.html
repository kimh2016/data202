The dots or the points are presented in green and there are no specific size is assigned. When the viewer moves the cursor on a certain point, the total rate of that country shows up. Wrangling

I will need to join religion and smoking data sets first because they have the full names of the countries. Then I will need to use gapminder::country_codes to add full names to gdp data set which only has country abbreviations. Finally, I will join two dat sets that will have both full names of countries and abbreviations, gdp, and religion.

## Rename columns and select useful columns of the gdp data set

gdp <- rename(gdp, gdp = Value, country = LOCATION) 
gdp <- gdp %>% 
  select(country, TIME, gdp)
## Join smoking and religion data sets. 

smoking_new <- smoking %>% 
  select(country, rateBoth)

smoking_religion <- religion %>% 
  left_join(smoking_new) %>%  
  drop_na()
## Joining, by = "country"
# make country names same as the ones in gapminder::country_codes 

smoking_religion[smoking_religion == "South Korea"] <- "Korea, Rep." 
smoking_religion[smoking_religion == "North Korea"] <- "Korea, Dem. Rep."
## Join country abbreviations names to the gdp data set using gapminder::country_codes

#View(gapminder::country_codes)
gapminder <- gapminder::country_codes %>% 
  select(country, iso_alpha)

smoking_religion <- left_join(smoking_religion, gapminder, 
                               by = c("country" = "country")) %>% drop_na()
## Finally, combine all three data sets into a one data set called semi_final. 

semi_final <- left_join(smoking_religion, gdp, by = c("iso_alpha" = "country"))
## Filter countries by the most recent year (2020).

final <- semi_final %>%
  group_by(country) %>%
  filter(TIME > 2019) %>%
